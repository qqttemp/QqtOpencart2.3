=========================================================
					INSTALLATION STEPS:
=========================================================

1) Place the Theme code:
	Paste the "fashion" Code inside your project_folder -> catalog -> view -> theme

2) Install the Fashion theme:
	 to install the theme, login to your admin panel and go to Extensions -> Extensions 
	 Select Themes from dropdown
	 Now select fashion from Theme Directory dropdown and save the changes

3) Activate the fashion theme:
	To activate the theme, login to your admin panel and go to system -> setting and edit Your Store then select the Fashion Store Theme from  the theme dropdown

4) Enable latest, special, Featured, Bestseller modules:
	Install and enable the latest and special modules from admin pannel Extensions -> Extensions 
	Select Themes from dropdown
	
5) Change the Home page Layout:
	From admin pannel go to, system -> design -> layout from menu
	Edit the Home
	Assign all your module in the position of Content bottom by sort order, except the slideshow > homepage
	Leave the slideshow > homepage module in the positon of Content Top i.e as it is!
	
	Example Module Details for Home Page:

		Module			position		sort order

		slideshow > homepage	Content Top		1
		featured  > homepage	Content Bottom		1
		latest    > homepage	Content Bottom		2
		special   > homepage	Content Bottom		3
		bestsellers>homepage	Content bottom		4
		carousel  > homepage	Content Bottom		5

6) Configure the Logo and fav icon:
	Change your theme logo and fav images from admin pannel system -> settings -> image tab

7) Setting the images size:
	For configuring the images size, login in to the admin pannel
	go to Extensions -> Extensions, Select Themes from dropdown, edit the theme and change the following image specifications!
	
	Title of the image		width		height
	Category Image Size		1500		300
	Product Image Thumb Size	450		550
	Product Image Popup Size	500		600
	Product Image List Size		300		400
	Related Product Image Size	240		320

	For setting module images size:
	
		Extensions -> Extensions, Select Modules from dropdown in admin pannel menu, and change the following modules image size as follows:
			
			Module Name		Width		Height			
			Featured		240		320
			Latest			240		320
			Special			240		320
			Bestsellers 240		320
